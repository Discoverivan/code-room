# Security Policy

## Supported versions

Security fixes are released for the latest published version.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability.

Use GitHub's private vulnerability reporting for this repository:

https://github.com/Discoverivan/code-room/security/advisories/new

Include reproduction steps, affected versions, and potential impact. Please do
not include real room links or private document contents.

Room links are access secrets. Deployments should use HTTPS and keep `/data`
private and backed up.
